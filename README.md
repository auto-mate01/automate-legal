# AutoMate GitHub Pages Site

Bu klasör GitHub Pages ile doğrudan yayınlanabilecek statik sitedir.

## Yayınlanacak sayfalar

- `/privacy-policy/` -> Türkçe gizlilik politikası
- `/en/privacy-policy/` -> English privacy policy
- `/account-deletion/` -> Hesap ve veri silme talebi sayfası

## GitHub Pages hızlı kurulum

1. GitHub'da yeni bir public repository oluştur: `automate-legal`
2. Bu zipin içindeki dosyaları repository kök dizinine yükle.
3. Repository içinde `Settings > Pages` bölümüne gir.
4. Source olarak `Deploy from a branch` seç.
5. Branch: `main`, Folder: `/root` seç ve kaydet.
6. Birkaç dakika sonra site şu formda yayınlanır:
   `https://<github-kullanici-adin>.github.io/automate-legal/`

## Play Console için URL

Gizlilik politikası URL'si:
`https://<github-kullanici-adin>.github.io/automate-legal/privacy-policy/`

Hesap silme URL'si:
`https://<github-kullanici-adin>.github.io/automate-legal/account-deletion/`

## Yayınlamadan önce

Gizlilik politikası içindeki geliştirici iletişim e-postasını resmi destek e-postanla değiştir.
